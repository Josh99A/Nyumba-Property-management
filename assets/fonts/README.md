# Bundled document fonts

Nyumba embeds these fonts so generated PDFs support all four application
languages, including correctly shaped Arabic text:

- Noto Sans variable font: https://github.com/notofonts/latin-greek-cyrillic
- Noto Naskh Arabic variable font: https://github.com/notofonts/arabic

Both fonts are distributed under the SIL Open Font License 1.1. The matching
license texts are stored beside the font files.
